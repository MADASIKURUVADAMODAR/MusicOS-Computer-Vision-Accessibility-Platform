# 🎵 MusicOS – AI Gesture Controlled Accessibility Platform

Real-time computer vision system enabling touchless music control through hand gesture recognition, voice commands, and accessibility-focused interaction.

## Features

- Hand Gesture Recognition using MediaPipe
- Voice Commands
- Real-time Webcam Processing
- Music Playback Controls
- Favorites Management
- Speech Recognition
- Accessibility Focused Design

## Tech Stack

- React
- Python
- MediaPipe
- Speech Recognition
- Computer Vision
- JavaScript
